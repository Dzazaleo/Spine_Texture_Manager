```

```

Key invariant from libgdx convention (RESEARCH §"Sharp Rotation Direction"):
`rotate:true` means the packer rotated the source 90° CCW when packing. So
`region.degrees == 90` will appear in spine-core's parsed TextureAtlasRegion
(verified at node_modules/@esotericsoftware/spine-core/dist/TextureAtlas.js:87-93).
