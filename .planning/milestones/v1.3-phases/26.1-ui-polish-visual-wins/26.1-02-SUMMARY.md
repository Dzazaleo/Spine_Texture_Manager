## Decisions Made

- Danger tint takes exclusive priority over zebra via ternary (not `&&` additive) — a missing row never shows striping, intentional per D-06 + D-10 spec
- AnimationBreakdownPanel already had `state` prop in scope (BreakdownRowItemProps), so the same `state === 'missing'` condition works identically to GlobalMaxRenderPanel
- SVG triangle uses `fill="currentColor"` not `stroke` — DimsBadge uses stroke for the info-circle, but a filled solid triangle is more conventional for warning indicators; wrapper dimensions are identical (w-4 h-4)
