## Shared Patterns

### SVG icon at `w-4 h-4` (16×16px)
**Source:** `src/renderer/src/components/DimsBadge.tsx` lines 72–91
**Apply to:** C-04 (warning icons in both panels), C-08 (danger panel header icons)
```tsx
<span
  className="inline-flex items-center justify-center w-4 h-4 ml-1 align-middle text-warning"
>
  <svg
    viewBox="0 0 16 16"
    stroke="currentColor"
    strokeWidth="1.5"
    fill="none"
    className="w-4 h-4"
    aria-hidden="true"
  >
    ...path...
  </svg>
</span>
```
Key: wrapper uses `inline-flex items-center justify-center w-4 h-4`; SVG gets `className="w-4 h-4"` and `aria-hidden="true"`. For fill-style (triangle) icons, use `fill="currentColor"` and omit `stroke`/`strokeWidth`.

### `clsx` conditional class pattern — Tailwind v4 literal discipline
**Source:** `src/renderer/src/panels/GlobalMaxRenderPanel.tsx` lines 416–422
**Apply to:** C-03/C-07 `<tr>` in both panels
```tsx
className={clsx(
  'border-b border-border hover:bg-accent/5',
```
