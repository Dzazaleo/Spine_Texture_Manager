## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| developer machine → produced installer | YAML config controls every byte that ships to testers; misconfig is the primary security surface |
| installer → tester machine | unsigned/ad-hoc binaries cross OS-level Gatekeeper / SmartScreen; tampering in transit is undetectable without a real signature |
| installed app → sharp libvips dylib | asarUnpack patterns control whether dlopen() finds the right native binary; broken asarUnpack = broken Optimize Assets |
