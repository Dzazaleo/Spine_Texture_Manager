## What Was Built

### Task 1 — UnusedAssetsPanel refactor

- **Props interface** expanded: `query: string` and `onQueryChange: (q: string) => void` added (matching GlobalMaxRenderPanel shape, per D-03 single-SearchBar contract).
- **Local query state removed**: `const [query, setQuery] = useState('')` deleted. `filteredOrphans` useMemo now reads the `query` prop directly — dependency array `[orphanedFiles, query]` unchanged.
- **SearchBar removed**: import and the `div.basis-full mt-2 SearchBar` JSX block at former lines 87-93 deleted. The AppShell toolbar SearchBar is the sole search surface.
- **Icon converted to stroke**: SVG changed from `fill="currentColor" viewBox="0 0 16 16"` to `fill="none" stroke="currentColor" strokeWidth="1.5" viewBox="0 0 20 20"` with triangle + exclamation path+line+circle.
- **Tests updated**: all 9 render calls now pass `query=""` and `onQueryChange={() => {}}`. Test (i) replaced `fireEvent.change` on internal textbox with `rerender` prop-drive approach.

### Task 2 — AppShell 3-tab restructure

- **ActiveTab type**: `'global' | 'animation'` → `'global' | 'unused' | 'animation'`.
- **D-02 auto-redirect useEffect**: added near the onClickAtlasPreview / setActiveTab call sites. Watches `(effectiveSummary.orphanedFiles ?? []).length` (primitive) + `activeTab` — redirects to `'global'` when Unused tab is active but orphaned files cleared.
- **TabButton enhanced**: `icon?: ReactNode` and `badge?: number` optional props added. Layout changed from bare text to `inline-flex items-center gap-1` with icon left, label center, badge pill right. Active branch changed from `font-semibold` to `font-medium` per UI-SPEC 2-weight limit.
- **Tab strip updated**: Global gets `icon={<GridIcon />}`, Unused conditionally rendered when `orphanedFiles.length > 0` with `icon={<WarningTriangleTabIcon />}` and `badge={count}`, Animation gets `icon={<ChartBarsIcon />}`.
- **Icon helpers defined**: `GridIcon`, `WarningTriangleTabIcon`, `ChartBarsIcon` — all stroke style, `viewBox="0 0 20 20"`, `w-4 h-4`, placed after the TabButton function.
- **UnusedAssetsPanel gated**: moved from always-rendered to `{activeTab === 'unused' && (<UnusedAssetsPanel ... query={query} onQueryChange={setQuery} />)}`.
