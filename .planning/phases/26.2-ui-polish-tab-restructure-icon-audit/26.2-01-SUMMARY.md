## AP-01 Acknowledgement

> AP-01 is the toolbar-inlined tab anti-pattern. It manifested as two reverted
> attempts (commit `c13ca9c` + earlier) that placed the tab buttons inside the
> main toolbar `<header>` row alongside the file badge, counts, SearchBar,
> atlas-less checkbox, and action buttons. The structural mechanism that
> prevents recurrence is sketch-001 variant A's locked layout: a dedicated
> `<nav role="tablist">` element rendered as a sibling row below the main
> toolbar `<header>`, separated by `border-t border-border`, never as a
> descendant of the toolbar.

The grep gate `awk '/<header /,/<\/header>/' src/renderer/src/components/AppShell.tsx | grep -c '<nav role="tablist"'` enforces this invariant in CI; current value is **0** (PASS).

## Accomplishments

- **AP-01 structurally resolved.** The `<nav role="tablist">` element is now a sibling of the main toolbar `<header>`, inserted immediately after `</header>`, separated by `border-t border-border`. Two prior reverts (c13ca9c + earlier) cannot recur without flunking the grep gate.
- **TabButton signature extended.** Optional `icon?: ReactNode` prop slot added at AppShell.tsx:1741. Renders the icon left of the label with `gap-1` (4px). All existing semantics preserved — 2-weight contract (`font-semibold`/`font-normal`), 2px orange underline indicator (`-bottom-px h-[2px] bg-accent`), ARIA tab role.
- **Tab icons added (verbatim panel-header SVG reuse).** Global tab renders the filmstrip from `GlobalMaxRenderPanel.tsx:836-839` (`<rect x="2" y="6" width="16" height="8" rx="1"/>` + 5 vertical tick marks). Animation Breakdown tab renders the bar-chart from `AnimationBreakdownPanel.tsx:415` (`<path d="M3 17 V10 M8 17 V6 M13 17 V12 M18 17 V4"/>`). Both at `w-4 h-4` (smaller than the `w-5 h-5` panel-header rendering, identical path data).
- **D-04 canonical stroke attributes uniformly applied** to both tab icons: `stroke="currentColor"`, `strokeWidth="1.5"`, `strokeLinecap="round"`, `strokeLinejoin="round"`, `fill="none"`, `viewBox="0 0 20 20"`, `aria-hidden="true"`.
- **D-02 type stability.** `ActiveTab = 'global' | 'animation'` unchanged. No `'unused'` literal anywhere in `AppShell.tsx` — confirms the 3-tab scope drop.
