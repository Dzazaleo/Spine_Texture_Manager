  **Status (2026-04-28):** Empirically falsified during Plan 12.1-01 Task 3 pre-flight. - **D-09 (supersedes D-01, locked 2026-04-28):** Use Option A from `12.1-01-BLOCKED.md` — change `electron-builder.yml` line 18 from `publish: null` to `publish: never`. Single-character edit. Hypothesis: electron-builder's `publish: never` mode constructs publish artifacts (including `latest-*.yml` feed files) without auto-uploading them, distinct from `publish: null` which suppresses publisher configuration entirely. The static `build/app-update.yml` + `extraResources` mapping from `679cffc` stays in place; the runtime feed-config source-of-truth remains the bundled YAML.

  Rejects Option B (restore `publish: github`) because it is the exact config that burned the 3 prior CI runs (25017095851 / 25017351602 / 25017624868); BLOCKED.md's framing of B as "Phase-11-working" conflates Phase 11 (no `latest-*.yml` requirement) with Phase 12 (added the requirement, hit the per-artifact publisher race). Rejects Option C (post-build synthesizer script) at this stage — it is the documented fallback if Option A's pre-flight fails empirically, but adding ~30 LOC of new script surface up front is premature complexity. Rejects Option D (`provider: generic` + hosted feed URL) — heaviest rework, requires new GitHub Pages surface, changes runtime semantics; reserved for if both A and C fail.

  **Validation gate (mandatory before any code commit):** Re-run Plan 12.1-01 Task 3 pre-flight locally — `npm run build:mac`, then verify (a) bundled `app-update.yml` is byte-identical to `build/app-update.yml`, AND (b) `release/latest-mac.yml` exists post-build with valid sha512/size/version fields. If `latest-mac.yml` does not appear or is malformed → halt; do NOT commit; re-enter `/gsd-discuss-phase 12.1` to escalate to Option C; record the new empirical evidence in a follow-up BLOCKED-style document.

  **Fallback ordering if D-09 fails empirically:** A → C → D. Skip B entirely.

- **D-03:** rc2 → rc3 sequencing is ATOMIC across hosts, not per-OS incremental.
