**Resolution (locked here so the executor does not relitigate):** the ref is an OBJECT-shaped ref parallel to `appShellMenuRef`, with two members:

```typescript
dirtyCheckRef = useRef<{
  isDirty: () => boolean;
  openSaveQuitDialog: (onProceed: () => void) => void;
```
