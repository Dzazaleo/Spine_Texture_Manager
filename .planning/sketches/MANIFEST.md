# Sketch Manifest

## Design Direction

Dark, dense, tool-forward. Warm stone palette with orange accent (Spine-adjacent).
All chrome is understated so data tables dominate. Tabs exist to navigate panels,
not to decorate the UI. Icons must be the single source of truth — whatever SVG
lives in a panel header should be reused verbatim on that panel's tab button.

## Reference Points

- Phase 26.1 / 26.2 existing app chrome (stone-900 bg, orange-500 accent)
- Real token values from `src/renderer/src/index.css`
